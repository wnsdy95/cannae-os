# AAR Readiness Update Fixtures

These fixtures verify that AAR learning is converted into readiness, maintenance, SOP, and CCIR follow-up actions.

Expected behavior:

- normal improvement AAR -> S6 hold/train recommendation with verification actions.
- critical unsupported-source AAR -> Red Team downgrade/hold recommendation and commander review.
- sustain-only AAR -> sustain/raise recommendation without commander review.
