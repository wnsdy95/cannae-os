# Support

Cannae OS is an open-source prototype. Support is community best-effort.

## Where To Ask

- Use GitHub Issues for bugs, documentation defects, and concrete proposals.
- Use Discussions if enabled for broad questions or design exploration.
- Use Security reporting for sensitive vulnerabilities or secret exposure.

## Before Opening An Issue

Please include:

- the affected document, schema, runner, fixture, or skill;
- what you expected;
- what happened;
- validation commands you ran;
- relevant error output;
- whether the issue affects authority, release, risk, security, or source traceability.

## Out Of Scope

Maintainers cannot provide:

- legal, medical, financial, military, or safety-critical advice;
- production deployment guarantees;
- confidential review through public issues;
- custom implementation support without a concrete repository issue.
